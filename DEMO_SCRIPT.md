# Live Demo Script

## Best Demo Prompt

Use this first:

```text
We present tomorrow at 11 AM. Create missing prep tasks, block 2 hours for rehearsal, summarize the judging notes, and prepare a final checklist.
```

## 90-Second Flow

1. Open the app and say:
   "This is Chief of Staff AI, our multi-agent execution system for high-stakes workflows."

2. Point to the architecture proof cards and say:
   "The primary orchestrator coordinates three sub-agents for tasks, scheduling, and knowledge."

3. Point to Google Calendar and say:
   "We also connected a real external tool, so this is not just mock data."

4. If needed, click `Reset Demo Data`.

5. Click `Launch Workflow`.

6. While it runs, say:
   "The orchestrator is translating one request into a multi-step plan, then delegating work through MCP-style tools."

7. Point to the Executive Brief and say:
   "Instead of raw tool output, the system returns an executive-ready action summary."

8. Point to the Delegation Flow and MCP Audit Trail and say:
   "Here you can see which sub-agent acted, which tool it used, and the persisted audit trail."

9. Point to the workspace state and say:
   "The database is updated in real time across tasks, calendar, notes, workflow history, and tool logs."

10. If you want the strongest proof moment, click `Create Real Demo Event` and say:
    "This creates a real Google Calendar event from inside the workflow system."

## Judge Q&A One-Liners

- If they ask why this is multi-agent:
  "Because one coordinator plans the mission and delegates responsibility to specialized agents that each own a tool domain."

- If they ask where MCP is used:
  "The agents never talk directly to storage or services. They invoke capabilities through a shared MCP-style gateway for tasks, calendar, and notes."

- If they ask about persistence:
  "Every task, event, note, run, and tool call is stored in the database, so the system has memory across sessions."

- If they ask about real-world value:
  "This is an AI Chief of Staff for execution-heavy teams under deadlines, where coordination matters more than content generation."

## Last Line

End with:

"Chief of Staff AI shows that agent systems become valuable when they coordinate execution across tools, not just generate text."
